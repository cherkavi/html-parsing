Пакет, который содержит необходимые классы (интерфейсы) по работе с парсингом страниц:
	- получения информации обо всех страницах сайта
	- получения с искомой страницы блока данных
	- получения из блока данных очередной порции данных
	
	
[class WalkPage] содержит необходимые методы по прохождению всех страниц сайта
	HtmlPage getNextPage() - получить следующую страницу
	(class HtmlPage - представление Html страницы )
	
	
	[class WalkRecord] - содержит метод по выявлению записей для сохранения из блока данных ( предварительно его выявив )
		HtmlRecord getNextRecord()
		(class HtmlRecord - представление Html записи, из которой можно получить данные для записи )
